# Proyecto App Panadería

Este directorio está destinado para almacenar el proyecto de la aplicación de panadería.

## Instrucciones para subir archivos .zip

### Opción 1: Usando la interfaz web de GitHub
1. Navega a este directorio en GitHub
2. Haz clic en "Add file" > "Upload files"
3. Arrastra tu archivo `.zip` o haz clic para seleccionarlo
4. Escribe un mensaje de commit descriptivo
5. Haz clic en "Commit changes"

### Opción 2: Usando Git desde línea de comandos
```bash
# 1. Clona el repositorio (si no lo tienes)
git clone https://github.com/Marc0097/Python.git

# 2. Navega al directorio del proyecto
cd Python/Proyecto_App_Panaderia

# 3. Copia tu archivo .zip al directorio
cp /ruta/a/tu/archivo.zip .

# 4. Añade el archivo al staging
git add tu_archivo.zip

# 5. Crea un commit
git commit -m "Añadir proyecto de panadería"

# 6. Sube los cambios
git push origin main
```
